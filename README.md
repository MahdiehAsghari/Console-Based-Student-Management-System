# Console-Based-Student-Management-System

This is a simple Python command-line project that simulates a university course selection system.
It was built as a beginner project to practice working with menus, lists, and user input.

---

## Features

* Select courses from a list
* Add or remove selected courses
* Confirm your final choices
* Simple financial system (balance, payment, withdraw)
* Quit request and basic info section
* Colored terminal output using `rich`

---

## Requirements

* Python 3
* rich library

Install it with:

```bash
pip install rich
```

---

## How to Run

```bash
python main.py
```

Follow the menu in the terminal to use the program.

---

## Notes

* Data is not saved (resets after closing)
* This is a simple practice project, not a full system
